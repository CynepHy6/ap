# apla packager

Simvolio and Protypo files packager to import on ecosystem

## Usage of ./ap

  --conditions string
        conditions (default "ContractConditions(\"MainCondition\")")
  --input string
        path for input files (default ".")
  --menu string
        menu (default "default_menu")
  --output string
        output filename (default "out.sim.json")
  --prefix string
        prefix for pages and contracts
  -c shortcut for for --conditions (default "ContractConditions(\"MainCondition\")")
  -i shortcut for --input (default ".")
  -m shortcut for --menu (default "default_menu")
  -o shortcut for --output (default "out.sim.json")
  -p shortcut for --prefix
